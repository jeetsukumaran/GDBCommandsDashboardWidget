Program Stack

bt backtrace [n]
Print trace of all frames in stack; or of n frames.
frame [n]
Select frame number n frames, or frame at address n. If no n, display current frame.
up n
Select frame n frames up.
down n
Select frame n frames down.
info frame addr
Describe selected frame, or frame at addr.
info args addr
Describe arguments of selected frame.
info locals
Describe local variables of selected frame.
info reg [rn]
Describe register values (for registers rn).
info catch
Describe exception handlers active in selected frame.
