Execution Context

run
Start program with current argument list.
run [arglist]
Start program with arguments arglist.
show args
Display argument list.
set args arglist
Specify arglist for next run.
set args
Specify empty argument list for next run.
show env
Show all environment variables.
show env var
Show value of environmental variable var.
set env var string
Set value of environmental variable var to string.
unset env var
Remove var from environment.
